2群のパラメトリック検定

▪母集団から取り出した$n$個の標本の平均値の分布を考える

ここでは，個々の標本データの分布ではなく，$n$個の標本平均$\bar{X_i}$の分布を考える．下図のように，正規分布している母集団$N(\mu,\sigma^{2})$から$n$個の標本を取り出し標本平均$\bar{X_i}$を計算する．この手順を何度も繰り返して得られた標本平均を，一つの別の母集団と考え分布を調べると，正規分布し，その平均値は$\mu$に近づくことが知られている．
しかし，母集団の正規分布$N(\mu,\sigma^{2})$とはばらつき方が異なり，$1/n$だけ小さいばらつき方になることがすでにわかっている．

関係する用語の整理
  • 大数の法則：サンプルサイズ$n$を大きくしていくと，標本平均$\bar{X}$が母平均$\mu$に限りなく近づく
  • 中心極限定理：母集団がどのような分布であっても，サンプルサイズ$n$が大きいときに，標本平均$\bar{X}$の分布が正規分布で近似できる

▪正規分布を標準正規分布に

正規分布は，母平均$\mu$と母分散$\sigma^{2}$が違えば分布が異なるので，いろいろな母集団で異なる正規分布を考えるのは面倒である．そこで，$\mu=0$，$\sigma=1$の正規分布を標準正規分布と定義し，それぞれ異なる正規分布を標準正規分布に変換（標準化）して考える．
標準化は次の式で定義される． $$ Z_i = \dfrac{\bar{X_i}-\mu}{\dfrac{\sigma}{\sqrt{n}}} $$

母標準偏差$\sigma$はわからないので，不偏標準偏差$u$に置き換える． $u$に置き換えると，正規分布とは少し違う分布となり，t分布と呼ばれる（両裾が広い）． $$ t_i = \dfrac{\bar{X_i}-\mu}{\dfrac{u}{\sqrt{n}}} $$

▪標本平均±標準誤差（standard error; SE）

$t_i$式の分母である$u/{\sqrt{n}}$は標本平均がどのようなばらつき方をしているかを示しており，これが標準誤差である．$SE$は母平均$\mu$がどのあたりにあるのか，すなわち，母平均$\mu$のありそうな範囲を表している．また，$SE$の式を見ると，$n$が大きくなればなるほど，母平均$\mu$の存在する範囲が絞りこまれることがわかる．

~ 標本平均$\pm{SD}$と標本平均$\pm{SE}$の違い ~
  • 標本平均$\pm{SD}$
    母平均$\mu$と母標準偏差$\sigma$の両方に関心がある．
  • 標本平均$\pm{SE}$
    母平均$\mu$に関心が集中しており，母標準偏差$\sigma$には関心がない．

2群の差の検定

▪仮説検定の考え方

・帰無仮説・対立仮説

群間比較をおこなう検定において，「差がない」ことから考えていく考え方が基本であり共通している．この「差がない」という仮説を帰無仮説と呼び，「差がある」という仮説を対立仮説と呼ぶ．

・p値・有意水準$\alpha$

仮説検定では，帰無仮説が正しいと仮定する仮想世界を想定する．実際に得られたデータが，この仮想世界では非常に稀な場合，仮想世界が間違っている，つまり，仮想世界における「帰無仮説が正しい」という仮定が間違っていると考えられる．

  • p値：帰無仮説が正しいと仮定したときに，観察された値以上に極端な値が出る確率
  • 有意水準$\alpha$：帰無仮説を棄却するかどうかの判断の境目に用いる値 　

仮説検定の流れとして，まず母集団に関して帰無仮説と対立仮説を設定する．帰無仮説が正しい世界を想定し，実験や観察によって得た標本データから，その起こりやすさをp値で評価する．p値が有意水準$\alpha$を下回るかどうかで，帰無仮説の棄却を決める．

・2群の差の検定の種類

  • 対応のない2群の差の検定
    e.g. 独立した2群のネズミで，対照食と試験食を与えて比較する場合
  • 対応のある2群の差の検定
    e.g. 同じネズミで，食品成分投与前と後の血圧を比較する場合

＊等分散性の確認
等分散かどうかは，等分散性の検定を行い，等分散と判断されればunpaired t testを，等分散と判断されなければWelchの検定をおこなう．
また，Welchの検定は，等分散でない場合だけではなく，等分散かどうか判断できない場合でも正確な検定ができる．一方で，unpaired t testは，等分散から外れると検定が不正確になる．
加えて，等分散性の検定は一般的に$n=30$以上が必要になる．したがって，標本の大きさ$n$が小さい場合，等分散性の検定があてにならない．そうなると，$n$が小さいとき，unpaired t testをおこなうのは危険である．結局，標本の大きさ$n$にかかわらず，等分散性の検定はおこなわず，Welchの検定のみおこなえばいい．

▪対応のない2群の検定（unpaired t test）

2つの標本$\bar{X_1}, \bar{X_2}$の母平均をそれぞれ$\mu_1, \mu_2$とし，母分散を$\sigma_1^{2}, \sigma_2^{2}$とする．

  • 帰無仮説：差がない（$\mu_{1}=\mu_{2}$）
  • 対立仮説：差がある（$\mu_{1}\neq \mu_{2}$）

＊正規分布の性質から，$(\bar{X_1}-\bar{X_2})$も正規分布になることがわかっているに注意する．

上図の$t_1$値が$t$分布のどこに位置するかで，有意差を決める．下図の青い領域は，有意水準$\alpha$での棄却域といい，棄却域に現実の値が含まれるとき，${p}\lt{\alpha}$となり，帰無仮説を棄却することになる．ここで，p値は$|t|$の外側の面積であることに注意されたい．

▪Welchの検定

基本的な考え方は，unpaired t testとよく似ているが，分散が異なるあるいは，分散が未知の場合を考える．
