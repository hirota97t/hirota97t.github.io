2群のノンパラメトリック検定

▪ノンパラメトリック検定の特徴

・母集団の分布によらない検定
・平均値や分散などのパラメータは使用しない
・母平均にはあまり意味がないので，中央値で考える
・中央値がどのくらい離れているかで考える

▪ウィルコクソンの順位和検定の考え方

順位和検定の基本は並べ替えである．下図に，A群とB群にそれぞれ6個のデータがあり，それらのデータを小さい値から大きい値に向かって順に並べている．順位和検定では，この順位の数字を用いて検定する．したがって，データの数値は順位となるため，平均値も分散も関係なくなる．

①と②の12個のデータを順位どおりに並べたのが③である．このように，ノンパラメトリック検定では，データのばらつきは関係なくなり，等間隔で並ぶことおよび外れ値が見えなくなることが特徴である．
ここまでで，2群間に大きな差があれば，順位の合計の差も大きくなりそうということがわかる．もし差があまりないと，白丸と青丸が比較的交互に並ぶことになる．順位和検定の原理を以下に示す．

▪ノンパラメトリック検定の利点・欠点

・母集団の分布に神経質になる必要がない
・母集団が正規分布の場合，検出力がやや落ちる
・外れ値がある場合に，有効性が高いことが多い
・サンプルサイズが小さいと検定が難しくなる
